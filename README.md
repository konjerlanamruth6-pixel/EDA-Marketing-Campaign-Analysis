# SkillCraft Technology Internship – Task 4

# Marketing Campaign Exploratory Data Analysis (EDA)

## Project Overview

This project was completed as part of my **Data Analytics Internship at SkillCraft Technology**.

The objective of this task was to perform **Exploratory Data Analysis (EDA)** on a marketing campaign dataset. The analysis focuses on understanding customer behavior, evaluating campaign performance, identifying trends, and providing business recommendations on which marketing channels should receive more budget based on Return on Investment (ROI).

---

## Objectives

- Load and clean the marketing campaign dataset.
- Perform Exploratory Data Analysis (EDA).
- Analyze customer demographics and purchasing behavior.
- Evaluate campaign performance.
- Generate actionable business insights.
- Recommend marketing strategies based on data-driven findings.

---

## Analysis Performed

- Handling Missing Values

- Customer Demographic Analysis
- Spending Pattern Analysis
- Campaign Response Analysis
- Correlation Analysis
- Data Visualization

---

## Key Insights

- Identified customer segments with higher campaign response rates.
- Analyzed customer spending behavior across different product categories.
- Evaluated the effectiveness of marketing campaigns.
- Identified factors influencing customer purchases.
- Provided recommendations to improve marketing ROI and optimize campaign budget allocation.
